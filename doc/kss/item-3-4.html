3.4 #permalink Formularz drabinek 

Kontener na wysuwany formularz, zbierający dane o turnieju i jego uczestnikach.
Parameters:
  • .complex-form
    główny kontener formularza. Centrowanie, marginesy, czcionka i kolory + animacja wysuwania i chowania
  • .complex-form label
    stylowanie labelów w formularzu. Wyświeltanie blokowe, pogrubienie, kolor czcionki
  • .complex-form input,.complex-form select,.complex-form textarea
    ustawienia wyświetlania elementów formularza
  • .complex-form button
    pociemnienie guzików, i inne style guzików formularza
  • .complex-form button:hover, header button:hover, #collapseFormButton:hover
    animacje najechania na guzik
  • .complex-form .form-group
    flexbox i marginesy dla lepszego widoku formularza
  • .complex-form .form-group label
    margines po labelu
  • @keyframes expandForm
    animacja rozwijania formularza
  • @keyframes collapseForm
    animacja zwijania formularza
  • .expanded
    klasa z animacją rozwijania
  • .collapsed
    klasa z animacją zwijania
  • .complex-form.expanded
    klasa do dodania animacji przez Javascript do rozwijania formularza
  • .complex-form:not(.expanded)
    klasa do dodania animacji przez Javascript do zwijania formularza
  • .button,#collapseFormButton
    usunięcie obramowania, zaokrąglenie guzika, zmiana kursora
  • .button,#collapseFormButton,.complex-form button
    kolor czcionki guzików, mała animacja
  • #submitFormButton
    jaśniejszy kolor głównego guzika
  • #collapseFormButton:hover,.complex-form button:hover
    powiększenie guzika i rozjaśnienie czcionki po najechaniu kursorem
  • .form-buttons
    centrowanie guzików w formularzu
  • .form-buttons button
    przyciemnienie pozostałych guzików
  • .complex-form button
    główne ustawienia guzików formularza
  • .complex-form .form-group
    centrowanie guzików formularza
